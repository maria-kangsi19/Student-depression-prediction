
#  Student Depression Prediction

## Project Description

This project uses Machine Learning to predict whether a student is likely to experience depression based on academic, personal, and lifestyle factors. It covers the complete ML workflow, including data preprocessing, visualization, model training, and evaluation. Logistic Regression and Random Forest classifiers were used and compared, with Random Forest achieving the best performance.

##  Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

##  Evaluation

- Accuracy Score
- Confusion Matrix
- Classification Report
